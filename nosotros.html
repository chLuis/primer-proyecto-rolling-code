NOSOTROS

Pertenecemos al Grupo 1 de la comisión 35i de Rolling Code School, éste es el primer Proyecto integrador, en el cuál estamos utilizando las herramientas de HTML, CSS y Bootstrap. La temática para esta entrega decidimos que sea la del Mundial de fútbol en Qatar 2022. El proyecto cuenta con una página inicial que posee las últimas noticias, una sección de artículos, una sección de contacto y esta sección de nosotros. También posee un navbar, un modal de login y una página de registro.
